An interactive educational tool that visualizes sorting and searching algorithms, helping students understand complex algorithmic concepts through animation.

## Features

- **Algorithm Visualizations**:
  - Quick Sort
  - Merge Sort
  - Binary Search
- **Interactive Controls**: Step-through and speed adjustment
- **Responsive Design**: Works on all devices
- **Clean UI**: Built with TailwindCSS

## Educational Value

- Makes abstract concepts concrete and visual
- Helps students understand algorithm complexity
- Provides step-by-step execution visualization
- Suitable for computer science education
